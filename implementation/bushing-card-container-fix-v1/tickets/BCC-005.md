# BCC-005: Fix Right Column Container Dragging Issue

## Priority
**HIGH** - Right column drags entire container instead of individual cards

## Description
The right column cards (Drafting, Summary, Diagnostics) are wrapped in `BushingRightLaneCards` component with a `BushingSortableLane`, causing all 3 cards to move together instead of individually.

## Root Cause
File: `src/lib/components/bushing/BushingOrchestrator.svelte` lines 409-425

```svelte
<div class="flex flex-col gap-4 pb-8">
  <BushingRightLaneCards
    items={rightLaneItems}
    {dndEnabled}
    {form}
    {results}
    ... />
</div>
```

The `BushingRightLaneCards` wraps all cards in a single sortable container.

## Solution
Inline the right column cards directly into the orchestrator, similar to how left column cards are handled.

## Implementation

### Step 1: Remove BushingRightLaneCards wrapper

**File: `BushingOrchestrator.svelte`**

Replace the right column section with inline cards:

```svelte
<div class="flex flex-col gap-4 pb-8">
  <BushingSortableLane
    listClass="flex flex-col gap-4"
    laneType="bushing-top-right"
    enabled={dndEnabled}
    items={rightLaneItems}
    on:finalize={(ev) => commitRightLane(ev.detail.items)}
    let:item>
    {#if item.id === 'drafting'}
      <BushingDraggableCard 
        column="right" 
        cardId="drafting" 
        title="Drafting View"
        dragEnabled={dndEnabled}
        canMoveUp={canMoveInList(rightCardOrder, 'drafting', -1)}
        canMoveDown={canMoveInList(rightCardOrder, 'drafting', 1)}
        onMoveUp={() => moveRightCard('drafting', -1)}
        onMoveDown={() => moveRightCard('drafting', 1)}>
        <BushingDraftingPanel ... />
      </BushingDraggableCard>
    {:else if item.id === 'summary'}
      <BushingDraggableCard 
        column="right" 
        cardId="summary" 
        title="Results Panel"
        dragEnabled={dndEnabled}
        canMoveUp={canMoveInList(rightCardOrder, 'summary', -1)}
        canMoveDown={canMoveInList(rightCardOrder, 'summary', 1)}
        onMoveUp={() => moveRightCard('summary', -1)}
        onMoveDown={() => moveRightCard('summary', 1)}>
        <BushingResultSummary ... />
      </BushingDraggableCard>
    {:else if item.id === 'diagnostics'}
      <BushingDraggableCard 
        column="right" 
        cardId="diagnostics" 
        title="Diagnostics"
        dragEnabled={dndEnabled}
        canMoveUp={canMoveInList(rightCardOrder, 'diagnostics', -1)}
        canMoveDown={canMoveInList(rightCardOrder, 'diagnostics', 1)}
        onMoveUp={() => moveRightCard('diagnostics', -1)}
        onMoveDown={() => moveRightCard('diagnostics', 1)}>
        <BushingDiagnosticsPanel ... />
      </BushingDraggableCard>
    {/if}
  </BushingSortableLane>
</div>
```

### Step 2: Add helper functions

```typescript
const moveRightCard = (cardId: RightCardId, direction: -1 | 1) => {
  rightCardOrder = normalizeOrder(
    moveCardInList(rightCardOrder, cardId, direction), 
    RIGHT_DEFAULT_ORDER
  );
};

const commitRightLane = (items: Array<{ id: string }>) => {
  rightCardOrder = normalizeOrder(
    items.map((i) => i.id) as RightCardId[], 
    RIGHT_DEFAULT_ORDER
  );
};
```

### Step 3: Delete BushingRightLaneCards.svelte
This file is no longer needed.
