# FAS-031 — Thermal Superposition Toggle

## Objective
Add thermal strain/load superposition on axial stack.

## Required Equations
- `delta_th,i = alpha_i * DeltaT * L_i`
- For constrained stack: `F_th = - sum(delta_th,i) / sum(C_i)`

## Acceptance
- Toggle OFF => `F_th = 0` exactly.
- Toggle ON => deterministic signed thermal force contribution.
