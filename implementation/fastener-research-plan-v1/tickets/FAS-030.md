# FAS-030 — Torque-to-Preload Exact Solver

## Objective
Invert thread + bearing torque to preload using exact mechanics.

## Required Equations
- `lambda = atan(p / (pi d_m))`
- `phi = atan(mu_t / cos(alpha))`
- `T = F * [ (d_m/2) * tan(lambda + phi) + (mu_b d_b / 2) ]`
- `F = T / [ (d_m/2) * tan(lambda + phi) + (mu_b d_b / 2) ]`

## Acceptance
- Forward and inverse consistency validated numerically.
- No nut-factor approximation allowed in solver internals.
