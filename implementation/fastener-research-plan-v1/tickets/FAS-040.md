# FAS-040 — Multi-Bolt Row Formulation Baseline

## Objective
Implement axial row force redistribution from compatibility + equilibrium.

## Baseline Formulation
- Unknowns: rigid translation `u0`, rotation `theta`
- Bolt extension at station `x_i`: `u_i = u0 + theta x_i`
- Bolt force: `F_i = k_i u_i`
- Equilibrium:
  - `sum(F_i) = F_ext`
  - `sum(F_i x_i) = M_ext`

## Acceptance
- Residual checks for force and moment close to zero.
