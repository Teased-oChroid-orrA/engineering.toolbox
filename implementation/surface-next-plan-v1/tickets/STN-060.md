# STN-060: D3/SVG Renderer Architecture Hardening

## Scope
- Keep D3/SVG as the single renderer path.
- Strengthen scene layering and visibility controls.
- Standardize renderer state model for theme/isolation/perf mode.

## Acceptance Criteria
- Only D3/SVG renderer options are exposed in Surface UI.
- Scene layering remains deterministic across edits.
- Renderer state is persisted and restored correctly.

## Tests
- `svelte-check` clean.
- Render-layer snapshot checks across create/delete/isolate flows.

## Rollback
- Revert to previous D3 layering strategy while keeping single-renderer policy.
