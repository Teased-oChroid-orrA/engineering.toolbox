# STN-080: Renderer Telemetry + Safety Throttles

## Scope
- Add telemetry for draw/update rate, interaction latency, and overscan utilization.
- Apply safe throttles for hover/selection at high dataset sizes.
- Show non-blocking notice when safety throttles activate.

## Acceptance Criteria
- Safety thresholds activate automatically under stress.
- User can manually clear throttled state when load stabilizes.
- No data loss during throttle transitions.

## Tests
- Fault-injection test for stress path.
- Recovery test from throttled mode back to full mode.

## Rollback
- Disable automatic throttles; keep telemetry only.
