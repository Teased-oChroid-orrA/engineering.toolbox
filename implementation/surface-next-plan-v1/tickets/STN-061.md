# STN-061: Frontend Renderer State Contract

## Scope
- Define and enforce typed renderer state for D3/SVG-only mode.
- Include theme, visibility, isolation, and perf mode fields.
- Ensure state round-trips through workspace persistence.

## Acceptance Criteria
- State schema is stable and versionable.
- No stale/unused renderer fields remain.
- Persistence restore does not produce invalid UI states.

## Tests
- Type-level checks and persistence smoke tests.

## Rollback
- Keep persistence logic but remove newly introduced state fields.
