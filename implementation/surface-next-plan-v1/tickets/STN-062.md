# STN-062: Scene Parity Baseline (D3/SVG)

## Scope
- Ensure D3 scene rendering covers points, edges, surfaces, CSYS, and planes.
- Add digest and scene counters for regression tracking.

## Acceptance Criteria
- All core entity classes render through the same path.
- Digest/counter metadata updates after each mutation.
- No schema drift between scene model and renderer usage.

## Tests
- Mixed-entity scene fixture test.
- Digest stability test for deterministic snapshots.

## Rollback
- Disable digest while retaining scene counters.
