# STN-070: Creative Visual System (D3/SVG-native)

## Scope
- Define visual strategy:
  - depth fog
  - cinematic gradients
  - selection halos
  - animated line energy
- Implement theme-ready params and runtime control hooks.

## Acceptance Criteria
- Theme parameter map exists and is switchable at runtime.
- At least three visual treatments are active in D3/SVG path.
- Visual identity is distinct while preserving readability.

## Tests
- Snapshot comparison across themes.
- Performance sanity at default dataset.

## Rollback
- Disable animated effects while retaining static theme colors.
